# VITA-CoughSense
VITA – AI-powered asthma cough monitoring app for Android. Features DeepCoughNet v4 (3-class cough classification) and CoughGate v3 (5 demographically specialized ONNX models, 83.25% val_acc), on-device exacerbation risk scoring, and Huawei Cloud integration (OBS, AGC, HMS Push Kit, SMN). Built with Flutter &amp; MindSpore 2.8.0.
